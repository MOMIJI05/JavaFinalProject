# 自動化電影推薦系統

此系統可以幫助不知道要看什麼電影的人
能夠根據使用者喜好的電影推薦給使用者合適的電影，也能夠隨意挑選想看的電影類型

## 功能

此系統有3種推薦方式：
- 根據使用者已收藏的電影，推薦更多使用者可能感興趣的電影
- 若使用者沒有收藏電影，則推薦Top100的電影
- 使用者自行選擇想要看哪些種類的電影
